Definition list

Ordered List:
An ordered list is a list of items with numerical or alphabetical order assigned to the items. These lists may be used for things like recipes, steps in a tutorial, references and footnotes, rankings, or any other number of possible uses where order is important to the nature of the items in the list. To make an ordered list, use (ol) tags wrapped around (li) tags for each list item.
Unorderd List:
An unordered list is a list whose order is not important and as such, does not use numbers or letters to order them, instead displaying bullets to mark individual items. They are created using (ul) tags that wrap around a series of (li) tags.
